The Duplicates Scanner

Even tho Pinterest does tell you wether a pin is already on one of your boards, duplicates could still happen.
For instance when images get pinned two times, two users pinning the same image, creating two seperate pins.

The Duplicates Scanner can be found via the main menu under Duplicates -> Open Duplicates Scanner. The dialog can be closed at any time by clicking the I'm Done button at the lower right or the close button of the dialog.

Setting Scanner Options

The scanner has two settings, which will influence how duplicate images will get scanned.

  1. Similarity

    The slider, in the lower left of the scanner dialog, controls the sensitivity of the scanner.
    Higher values will make the scanner more sensitive, resulting in a less strict comparison.
    Lower values will make the scanner more restrictive, making the scanner pay more attention to detail.

    For the best results it's best to run the scanner multiple times adjusting the slider to make the results more reliable.

  2. Multithreaded Scanning

    By default the scanner will process multiple images at the same time, depending on the cores available on your machine. This greatly improves the scanner's speed, but also puts more load on your machine.

    Turn this setting off if you are experiencing performance issues when using the scanner.

Scan Results

Pressing Start Scan will start scanning each board directory for duplicate images.
You can stop this process at any time by pressing Stop Scan. Note that the scanner will finish the current board before aborting.
Closing the scanner dialog will also stop the scanner.

During the scan filenames will start to appear in the left pane.
These filenames are images of which duplicates have been found.

You can click on a filename in the left pane to view it's duplicates.
On the right a list of previews will appear. The first item is the image with the best quality. Any items below the first item are duplicates or similar to the first item.

You can select items you want to delete in the right pane by clicking on said item. Hold the CTRL key or SHIFT key and click items to select multiple.
After selecting the items you want to delete click Delete Selected to delete the selected items.
Warning: Deleting ALL items in the above list will also delete the original file!
