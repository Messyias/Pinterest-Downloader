Using Pinterest Downloader

Pinterest Downloader is able to download all the pins on your Pinterest boards.

Important Notice

Pinterest Downloader does not use a database or alike to store information on already downloaded files.
Instead it prepends a unique id to the start of all filenames.
You are free to rename any file, created by Pinterest Downloader, as long as this id stays intact.

Aquiring a Pinterest Api Access Token

In order to be able to retrieve your pins from Pinterest Pinterest Downloader needs an access token.
Below are the steps necessary to aquire a Pinterest Access Token.

  1. Log on to Pinterest.
  2. Navigate to the Token generator.
  3. Make sure to check read_public and write_public. The other checkboxes can be left alone.
  4. Click Generate Token.
  5. Copy the string of characters that appear in the text box.
  6. Paste this string into the Pinterest Api Access Token field of the Pinterest Downloader application.

Before Clicking start

Pinterest Downloader allows you to setup what to do during and after processing.

Field: Target Download Directory

This is the directory to which Pinterest Downloader will download your pins.
To change the directory simply click Change... and use the shown browser to point to the correct directory and click Open.

Check Box: Download Boards

Unchecking this option will not download any pins. This is applicable when you just want Pinterest Downloader to check the files in the target directory for the correct extensions.

Check Box: Scan Files Types

Pinterest has the tendency to use the incorrect file type for files.
For instance GIF files could be returned as JPEG files, rendering them unopenable or not-animated.
Leaving this check box checked will have Pinterest Downloader automatically check and correct these files.

Check Box: Open Target Directory on Completion

Leaving this check box checked will open the target directory on completion.

What exactly happens when I press start?

Assuming you are using the default settings and entered a valid access token, Pinterest Downloader will start doing the following:

  1. Download Boards
    1. Fetch information on your Pinterest boards
    2. For each board the following process occurs:
      1. Retrieve information on all pins on the board
      2. Create a directory, within the chosen target directory
      3. Start downloading all pin images to above directory, any already downloaded image will be skipped
  2. Check downloaded images for correct filetype
    1. List all files in the target directory
    2. Start checking each file and correct it's filename if nessecary
  3. Open the target directory in the OS's file browser
